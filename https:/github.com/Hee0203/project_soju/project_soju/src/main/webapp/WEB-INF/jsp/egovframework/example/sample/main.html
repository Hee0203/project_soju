ABOUT

최근 2030세대는 펀슈머(Fun + Consumer)로 이색소비를 원합니다. 따라서 나오고 있는 요즘 소주들은 컨셉소주로 과일 또는 브랜드와 콜라보되어 나온 소주를 말합니다. 종류는 ['에이슬' 시리즈], [처음처럼 X 빠삐코] 등 많습니다. 어떤 것이 시초인지는 알 수 없지만 과일소주 및 이색소주가 인기가 많아지면서 컨셉소주가 다양하게 출시되어지고 있습니다.
